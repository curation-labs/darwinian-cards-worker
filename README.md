# Darwinian Cards Worker

This blueprint materializes one Worker for the `darwinian-cards` project. Cards compose capabilities into that Worker; Workers do not stack.

The blueprint pins independently published Fal, Operator, and Notion Card releases. Each member uses a direct immutable Git release ref so a clean machine can resolve the complete Worker after discovering only this blueprint.

The blueprint owns no copied skills, MCP definitions, Mind content, or other member files.
